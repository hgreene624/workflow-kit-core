# Save (manual fallback)

The one-word safety verb for when auto-trigger missed and the user wants it filed now.

## Procedure
1. Take the content in focus (the thing just discussed or produced).
2. Hand to the Artifact Engine: classify against conventions, generate, file into the active project.
3. If the type is ambiguous, ask one plain question ("a note, or part of {project}?") and file.

## Rules
- This is the manual backstop to best-effort ambient triggering. It should almost never be needed; when it is, it must just work in one word.
