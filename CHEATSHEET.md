# Talk to it — the cheat sheet

Workflow Kit is **ambient**. You almost never run a command. You just talk, and it offers to file things for you. You say yes.

This page is the whole interface. Keep it open for your first week; you won't need it after that.

## The one rule

**Describe what happened or what you want, in plain words.** The kit notices when something is worth keeping, proposes saving it, and files it in the right project when you confirm. If it ever misses, say **"save this"** and it files immediately.

## Just say this

| You say | What happens |
|---|---|
| "I'm working on a new project for the spring catalog." | Starts a project (goal / steps / log) and works there. |
| "I had a meeting with Ana about pricing." | Offers to save a meeting note with attendees + actions. |
| "Jot this down: the vendor raised the deposit to 30%." | Saves a quick note in the current project. |
| "What should I work on?" | Reads everything and tells you the next ready thing. |
| "Where did I leave off?" | Catches you up from where you stopped. |
| "What's blocked?" | Shows what's waiting on something else. |
| "Remember: always cc me on owner emails." | Learns the rule and applies it from now on. |
| "Save this." | Files whatever you were just discussing, right now. |
| "closeout" | Wraps the session, catches anything unsaved, sets up next time. |

## The same things in Spanish

The kit works in your language. If you set it up in Spanish, just talk in Spanish:

| Tú dices | Qué pasa |
|---|---|
| "Estoy trabajando en un proyecto nuevo para el catálogo." | Crea un proyecto y trabaja ahí. |
| "Tuve una reunión con Ana sobre precios." | Ofrece guardar una nota de reunión. |
| "Anota esto: el proveedor subió el depósito a 30%." | Guarda una nota en el proyecto actual. |
| "¿Qué debería hacer ahora?" | Te dice lo siguiente que está listo. |
| "Acuérdate: siempre ponme en copia en los correos de dueños." | Aprende la regla y la aplica. |
| "Guárdalo." | Guarda lo que estabas hablando, ahora mismo. |
| "closeout" | Cierra la sesión y deja todo listo para la próxima. |

## What you do NOT have to do

- You don't pick a "document type" or a folder. The kit does.
- You don't learn a list of commands. There's one: `closeout`.
- You don't write a spec or a plan to start a project. Just say what you're working on.
- You don't save manually after every message. It offers as you go; "save this" is only the backstop.

## When you're lost

Type **"what can this do?"** or **"help"** and it explains, in plain language, what you can save and where things are.

## The mental model in one line

> You have a conversation. The kit quietly turns it into organized notes and projects, and remembers where you left off. You confirm; it files.
